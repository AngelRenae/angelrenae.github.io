Selectors

  1. Home
  2. CSS
  3. Selectors

A selector is the element on the HTML page you want to style. You can pick many selectors at the same time or you can just choose one. Adding classes and ids to your HTML elements can help you be more specific on what styles you want applied to each element. Some typical selectors are:

element

Element selectors are selected using just the name of the element. For example, to select an h1 element you would put “h1” as the selector, to select an image you would put “img”, and so on. This will select every occurrence of that element on the page, though, so if you have one h1 element you want styles different from another h1 element, it would be best to go up in specificity.

An element being selected.

#id

ID selectors allow you to style an element based on the id that has been assigned to it. You can pick an ID selector by using a hashtag (#).

An ID being selected.

.class

Class selectors pick the element or group of elements that have been assigned to that class. You can select a class by using a period (.).

A class being selected.

element#some-id or element.some-class

To select an element with a specific class or id, you would put the element followed by either a hashtag and ID or a period and class.

The child of an element being selected.

element, element, element

You can select a group of elements by separating them with commas and a space.

Multiple elements being selected at once.
