Common Tags

  1. Home
  2. HTML
  3. Common Tags

There are many HTML tags. Aside from the four mentioned in HTML Overview ( <!DOCTYPE>, <html>, <head>, and <body>), there are more that you can typically build your page from. In the <head> tag, you may see:

<meta>

<meta> is a tag that defines a piece of metadata for the web page. You can use this tag to set the viewport, add an author to the webpage, or a description. This will not show up directly on the webpage but can be used by the browser.

<title>

<title> is a tag used to add a page title. This is displayed as the text for the current tab.

<link>

<link> creates a relationship between the current document and whichever document is linked. This can be used to create a relationship between the current HTML document and a stylesheet or import things like fonts from Google Fonts. The body of an HTML page has many tags to choose from to customize the page. Some of these tags are:

<h1> through <h6>

<h1> through <h6> are header tags that can be used to add headings to the page, better dividing the content. <h1> is the largest while <h2> is the smallest. It’s good practice to use the tags in order, like using <h1> as your most important heading and then <h2> as the second most important heading opposed to jumping to <h3> or <h4> because you think the size is more appropriate. They can be manipulated using a stylesheet, so following the natural order is best for clarity.

<p>

<p> is the paragraph tag, and is where the normal body text of a webpage can be put. Everything between the opening and closing tags will be displayed on the page as paragraph content. The paragraph tag will not pick up a new line added within the HTML document using just the enter bar, so you need to add a break tag (</br>) wherever you want the new line to start.

<ul>, <ol>, and <li>

<ul>, <ol>, and <li> are the three tags used to create lists. <ul> stands for unordered list, <ol> is ordered list, and <li> is list item. When creating a list, you take either <ul> or <ol> and put however many <li> you wantbetween the opening and closing tags. The content between the opening and closing tags of <li> will be the content displayed on the bulletin.

<a>

<a> is a link. This tag will need the attribute href=”[your link]” to link to anything. Whatever is wrapped in this tag will be the link text. <img> <img> is an image tag, and is used to add images to an HTML page. The attributes needed for this tag are src=”[your image]” and alt=”[a description of the image]”. They can be adjusted using CSS.
