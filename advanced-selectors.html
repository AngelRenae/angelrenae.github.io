Advanced Selectors

  1. Home

There are many more selectors you can use to properly narrow down what element you are trying to style. Those include:

element > another-element

This selector will take all of the occurrences of another-element that are children of the element and style them.

Makes all links after the unordered list blue.

element + another-element

This selector will style the another-element that directly follows the element. If we have <h1> followed by three <p>, the <p> directly after the h1 would be the only one affected. Any children of element will not be affected, even if they are another-element

The next paragraph element after h1 will have no text decoration.

element ~ another-element

This selector styles all of the another-element that occur after element. Any of the another-element that are children of element wonâ€™t be affected.

Every element with the class container will be styled as long as it is after and not a child of h3.
