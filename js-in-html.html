Embedded JavaScript

  1. Home
  2. HTML
  3. Embedded JavaScript

Javascript can be embedded in an HTML page by adding a <script> tag to the end of the body tag.It’s best to add it to the end so that it loads in after the content of the HTML page has. Javascript can be used to manipulate different HTML tags and handle any data paged to the page by the user. If any extensive Javascript is going to be used, it should be put into its own document just like CSS. You can use the script tag to link a Javascript document by adding src =”[your js file]” as an attribute.

A script tag at the end of the body in a HTML document.
